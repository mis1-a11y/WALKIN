☁ Google Apps Script — Live Deploy

Apne Google Sheet se seedha yahan live data fetch karo, bina CSV download kiye.
Step 1 — Google Sheet kholein
Apni Jewellery showroom Google Sheet mein jayein aur menu mein Extensions → Apps Script click karein.
Step 2 — Neeche ka code paste karein
Apps Script editor mein pura code replace karke neeche wala paste karein:
function doGet(e) { const ss = SpreadsheetApp.getActiveSpreadsheet(); const sheet = ss.getSheetByName(e.parameter.sheet || 'Actual data'); if (!sheet) return ContentService.createTextOutput( JSON.stringify({error:'Sheet not found'})).setMimeType(ContentService.MimeType.JSON); const rows = sheet.getDataRange().getValues(); const headers = rows[0]; const data = rows.slice(1).map(row => { const obj = {}; headers.forEach((h, i) => obj[h] = row[i]); return obj; }); return ContentService .createTextOutput(JSON.stringify({data, sheet: sheet.getName(), rows: data.length})) .setMimeType(ContentService.MimeType.JSON); }
Step 3 — Deploy karein
Apps Script mein Deploy → New deployment click karein.
Type: Web app | Execute as: Me | Who has access: Anyone
Deploy karne ke baad Web app URL copy karein.
Step 4 — URL paste karein aur connect karein
URL enter karein aur Connect dabayein
